Syntax guide

The default prefix is "b ".

To use any command listed here, you must use the prefix at the beginning.


{ input }

Braces: input

( option 1 | option 2 )

Parentheses: choice

[ parameter ]

Brackets: optional parameter


Commands

help [{command}]

Shows the help menu.

badges [{user}]

Views a user's profile and shows their badges.

badge (add|give) {user} {string badge}

Attributes a new badge to a user.

badge (remove|delete|del) {user} {int badge}

Removes a badge from a user.

redeem {code}

Redeem a code for an offical badge.

invite

Shows information on how to invite the bot in a server.
