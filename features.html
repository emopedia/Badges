All features of Badges!

Here is a full list of the features for Badges.


Customization

Badges contains alot of customization to fit your server needs. Need a badge to show an offical server moderator? This bot has got it.

Support

Badges has a great support server with great Staff and Support members. Need help configuring your bot? Ask in the support server.

Practicality

Badges is not just for cosmetics. You can use Badges as a way to verify staff members, certain users or reward users for doing something special.

Fun

Badges is designed to make your server more friendly as people try to increase their badge collection.

Commands

b help [{command}]

Shows the help menu.

b badges [{user}]

Views a user's profile and shows their badges.

b badge ((add|give)|(remove|del[ete])) {user} {badge}

Adds or Removes badges from users.

b redeem {code}

Redeem a code for an offical badge.

b invite

Invite the bot for use in your own server.
